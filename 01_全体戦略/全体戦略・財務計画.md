## 施術 収益モデル（2026年5月確定版）

### セッション構造

| 区分 | 時間 | 価格 | 備考 |
|------|------|------|------|
| 新規（初回） | 60分 | 19,800円 | 問診＋施術・全新規患者 |
| プログラム患者 | 45分 | 買い切り内 | 重症改善コース通院中 |
| 既存一般（Layer A） | 30分 | 6,000円 | 高齢・地域・チラシ集客層 |
| ダイエットクライアント | 30分 | ダイエットプログラム内 | 2週間に1回・整体込み |

### プログラム（重症改善コース）

| 項目 | 内容 |
|------|------|
| 価格 | 220,000〜240,000円（買い切り・平均230,000円） |
| 期間 | 3ヶ月 |
| 施術回数 | 月4回 × 3ヶ月 = 12回 |
| 施術時間 | 45分/回 |
| 対象 | 腰痛・ヘルニア・脊柱管狭窄症・重症患者 |

> **1件成約 = 230,000円の現金収入（成約月に計上）**

### キャパシティ

```
営業時間：9:00〜21:00（昼休憩90分）= 実働630分/日
30分1枠 = 最大21枠 → 70%稼働 = 14枠/日
月営業日：17日（月・水・金・土）
月間最大枠：238枠 / 70%稼働目標：166枠
現状稼働率：約37%（88セッション/月相当）
```

### 2層価格戦略（客層別）

| 層 | 流入経路 | 価格 | 役割 |
|----|---------|------|------|
| **Layer A**（高齢・低所得・チラシ集客） | チラシ・紹介 | 6,000円/30分 | 稼働率底上げ・口コミ・地域信頼 |
| **Layer B**（重症・解決志向） | LP・Instagram・YouTube | プログラム230,000円 | 収益の主力 |

**背景：** チラシ集客層は高齢者・低所得が多く8,800円では成約率低下。6,000円で受け入れ稼働率を上げながら、LP経由の重症患者でプログラム収益を積み上げる。

### 稼働率連動 単価引き上げルール

| 稼働率 | 新規標準単価 |

| 現状 37% | 6,000〜8,800円（状況判断） |

## ダイエット事業計画（整体込みプログラム）

| 項目 | 内容 |

> **整体込みのため、ダイエット成約 = 整体枠も同時確保される。**

| ダイエットクライアント数 | 消費枠/月 | 238枠に占める割合 |
|---------------------|---------|----------------|
| 5人 | 10枠 | 4.2% |
| **10人** | **20枠** | **8.4%** |
| 20人 | 40枠 | 16.8% |

> 10人稼働でも238枠の8.4%。整体稼働率に実質影響なし。

## プログラム成約の核心

```
【初回60分の設計】
問診でゴール共有
→「○○さんの状態だと3ヶ月・12回で改善できます」
→ プログラム提案
→ 当日or翌日に230,000円成約

月の売上は「何件プログラムが成約できたか」でほぼ決まる。
```
