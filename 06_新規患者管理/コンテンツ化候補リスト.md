# コンテンツ化候補リスト

## 目的
新規患者の症例から「投稿・LP・Google口コミ」に使えるネタを管理する。
患者の許可が取れたものだけをコンテンツ化する。

---

## 記録フォーマット

```
### 症例〇〇〇
- 患者属性：〇〇代・〇〇性・〇〇（職業）
- 主訴：
- 来院前の状況：
- 原因（施術で分かったこと）：
- 変化・結果：
- 患者の言葉（そのまま）：「　　　」
- 使える媒体：（Instagram / Google投稿 / LP / LINE配信）
- 制作状況：（未着手 / 作成中 / 投稿済み）
- 許可確認：（済 / 未）
```
