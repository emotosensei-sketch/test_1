## 設計メモ

### 各設問の意図

| 設問 | 意図 | 問診トークへの活用 |
|------|------|--------------------|
| 【1】症状・発症時期 | 施術方針の確認 | 「いつ頃から？」の会話のきっかけ |
| 【2】日常への影響チェック | 患者が「自分ごと」と認識する | 「〇〇も困ってるんですね」と共感 |
| **【3】悪化シミュレーション** | **当事者意識を作る最重要設問** | 「それを避けるために今が大事」という流れへ |
| 【4】これまでの対処 | 他院との差別化・信頼構築 | 「うちが違う理由」の説明材料 |
| 【5】良くなったらしたいこと | 短期理想の言語化 | プログラム提案の動機づけ |
| 【6】ゴール＋期日 | プログラム提案の土台 | 「○ヶ月コース」提示の根拠 |

### Canvaで作るときの注意点

- A4・1枚に収める
- 設問と設問の間は余白を十分にとる（詰め込まない）
- 【3】の記述欄は特に広めに（2〜3行分のスペース）
- フォントサイズ：設問見出し14pt以上、記入欄12pt以上
- チェックボックス□は大きめに

### 現行版から削除したもの・理由

| 削除した項目 | 削除理由 |
|------------|---------|
| メールアドレス | LINE管理で十分 |
| 辛い時間帯（朝/日中/夜間） | 問診中に口頭確認で十分 |
| 病院での検査結果 | 重症者は問診中に詳しく口頭で確認 |
| 「当院を最後の治療院に」等のチェックリスト | 設問1・2・3に統合 |
| 「メンテナンスしたいですか？」 | 設問6のゴール確認で代替 |
