# 人生主導権 回復スコア診断

> 用途：初回セッション（施術後）のクロージングツール
> 設計思想：押さない。売らない。選ばせる。

---

## 採点方法

各項目を **0〜10点** で自己評価してください。
0＝最悪の状態　10＝理想状態　**合計100点満点**

---

## 10項目

| # | 質問 | 基準 | 点数 |
|---|------|------|------|
| ① | 今、痛みは安定していますか？ | 0=毎日不安定 / 10=ほぼ気にならない | |
| ② | 朝起きた瞬間、不安なく動けていますか？ | 0=毎朝怖い / 10=全く不安がない | |
| ③ | 8時間活動しても悪化しませんか？ | 0=確実に悪化 / 10=問題なし | |
| ④ | 違和感が出ても48時間以内に戻せますか？ | 0=戻せない / 10=完全に戻せる | |
| ⑤ | 以前と比べて動きは改善していますか？ | 0=かなり制限あり / 10=制限なし | |
| ⑥ | 活動後の疲労はどうですか？ | 0=ぐったり / 10=ほぼ問題なし | |
| ⑦ | 「また悪くなるかも」と思う頻度は？ | 0=毎日思う / 10=ほぼ思わない | |
| ⑧ | 薬や湿布に頼っていますか？ | 0=常に使用 / 10=完全不要 | |
| ⑨ | 自分の悪化パターンを説明できますか？ | 0=全く分からない / 10=明確に説明できる | |
| ⑩ | 今、やりたいことを普通に選べていますか？ | 0=痛み基準で決めている / 10=気にせず選べている | |

## 点数の意味

| スコア | 状態 |
|--------|------|
| 0〜39点 | 痛みに人生の主導権を奪われています |

| 60〜79点 | 改善途中。再設計が必要 |
| 80〜100点 | 卒業圏内 |

## セッションでの使い方

1. 施術後のビフォーアフター確認のあとに書いてもらう
2. 合計点を一緒に確認する
3. **沈黙**
4. 一言だけ言う：

> **「この差を埋める設計をしますか？」**

**終わり。押さない。売らない。選ばせる。**
