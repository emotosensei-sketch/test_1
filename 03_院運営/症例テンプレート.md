# 症例テンプレート

## 投稿用テンプレート（Instagram・Google・LP共通）

```
【タイトル】
〇〇代・〇〇性・〇〇（職業）
→ 〇〇が〇ヶ月で改善した症例

【フック（1行目）】
「〇〇で悩んでいた方に見てほしい症例です」

【患者の状況】
〇〇の症状が続き、
「〇〇（患者の言葉）」
という状態でした。

【原因】
お体を確認すると、
〇〇の歪みによって〇〇に負担がかかり、
〇〇の状態になっていました。

【変化・結果】
1回の施術で〇〇を整えると、
✅ 〇〇が軽減
✅ 〇〇が改善
「〇〇（患者の言葉そのまま）」

【締め・哲学】
〇〇（専門家としての考え方・根本改善の視点）

【CTA】
同じようなお悩みの方、
プロフィールのリンクからご相談ください。
```

## キャプション構成のポイント
1. 冒頭：症状＋結果を1行で（読み飛ばし防止）
2. 患者の状況：感情・背景を具体的に
3. 原因：専門用語に簡単な説明を添える
4. 変化：✅で視覚化
5. 哲学：根本改善の考え方（専門性）
6. CTA：1択に絞る
7. 区切り線（━）で読みやすく

## ハッシュタグの付け方
- 毎回固定：`#寝屋川整体`
- 残り：その投稿の症状・内容に合わせて変える
- 目安：15〜20個
- 構成：症状タグ×地域タグ×技術タグ

## AIで量産するフロー
```
1. 患者情報を箇条書きで入力（年齢・症状・職業・結果）
2. Claudeがテンプレートに沿って生成
3. 確認・微修正
4. Google投稿・LP・Instagram・LINE 4媒体分を同時生成
```
