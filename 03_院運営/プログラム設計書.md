# 根本改善プログラム 設計書

> 作成：2026年3月（先生設計） / GitHub記録：2026年5月19日
> 価格：198,000円（確定） / 初回90分 + 45分×16回 = 18回

---

## コンセプト

**人生を取り戻す整体院**
― どこへ行っても改善しなかった慢性腰痛専門 ―

---

## 核心定義

| 定義 | 内容 |
|------|------|
| **治るの定義** | 痛みに人生の決定権を渡さない状態 |
| **人生主導権の定義** | 痛みを理由に諦めない人生に戻ること |
| **卒業の定義** | やりたいことを普通に選べる状態 |
| **商品の正体** | 腰を治すプログラムではない。痛みに奪われていた人生の決定権を取り戻す再設計プログラム |

> 核メッセージ：**「痛みが取れた＝治った」ではありません。**

---

## 3層構造

### 第1層：身体の再起動（1〜8回）
**目的：安心を身体にインストールする**

| 基準 | 内容 |
|------|------|
| 痛みスケール | 10 → 4以下 |
| 朝の状態 | 「今日は大丈夫かも」と思える日が週3回以上 |
| 活動 | 週3日以上、痛みを意識しない日がある |
| 可動域 | 30〜40％改善 |
| 思考 | 「治らないかも」が「戻れるかも」に変化 |

### 第2層：構造の書き換え（9〜12回）
**目的：再発しない土台を作る**

| 基準 | 内容 |
|------|------|
| 痛み安定性 | 1週間スケール3以下維持・強い再燃（6以上）なし |
| 活動量 | 日常活動量120〜150％・避けていた動作を3つ再開 |
| 再発理解 | 再発トリガーを言語化できる |
| 自己修正力 | 違和感→48時間以内に戻せる |
| 思考変化 | 「またぶり返すかも」週1回以下・未来不安スケール半減 |

### 第3層：人生の再設計（13〜18回）
**目的：痛み基準から人生基準へ戻す**

| 基準 | 内容 |
|------|------|
| 負荷テスト | 旅行 or 長時間外出実行・繁忙週を問題なく乗り切る |
| 行動基準 | 予定を痛み基準で決めていない |
| 自立管理 | 違和感→48時間以内自己修正・セルフ管理習慣化 |
| 心理的主導権 | 未来不安スケール3以下・「また戻る」は消えている |

## 卒業基準（第18回）

- 朝に怯えない
- 行動を痛み基準で選ばない
- 不調が出ても48時間以内に戻せる
- 将来を悲観しない

**＝ やりたいことを普通に選べる状態**

## プログラム詳細仕様（2026年5月26日更新）

### 基本設定
- 施術時間：45分/回（初回のみ90分）
- 価格：198,000円
- 回数：18回
- 打ち出し価格：初回2,980円

### 付帯サービス
- **インソール**：特典として提供
- **2回無料保証施術**：プログラム終了後も状態が戻った場合に対応

### 状態管理ツール
- 毎回施術後にスコアシート記入
- 10項目100点満点で変化を可視化
- 患者自身が「治っていく実感」を持てる設計
