# 運用ルール

## このリポジトリについて
- このリポジトリは整体院運営の**第二の脳**として育てていく
- 整体院に関する情報を蓄積していく場所として使う

## ブランチ運用ルール
- **mainを更新したら全てのブランチに自動で反映する**
- **新しいブランチが増えても同様に全て反映する**
- mainは常に最新・最正確な情報を保つマスターブランチとする
- 各ブランチはテーマ別の詳細情報を格納する

## ブランチ構成
| ブランチ | 用途 |
|---------|------|
| main | 全体ハブ・KPIダッシュボード・最優先タスク |
| branch-a | Instagram・LINE戦略の詳細 |
| branch-b | MEO戦略の詳細 |
| （追加予定） | 新テーマが増えたら随時追加 |

## 更新ルール
- 新しい戦略・気づきが生まれたら該当ファイルに追記する
- KPIが変わったら第二の脳メイン.mdのダッシュボードを更新する
- タスクが完了したら「未」→「✅」に変更する

## 表記統一ルール
- 「体」→「**身体**」に統一する

## Claudeとの連携
- 次の会話開始時は「第二の脳メイン.mdを読んで続きから始めて」と伝える
- 会話で決まった内容はその都度GitHubに保存する
