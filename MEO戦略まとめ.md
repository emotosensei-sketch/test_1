# MEO戦略まとめ
> 作成日：2026年4月27日

---

## 基本方針

- 見てくれた人が安心・信頼できる情報を届ける
- AIコンテンツが氾濫する中で「リアル」で差別化する
- 大手の広告費に対抗するため「量」ではなく「深さ」で戦う

---

## 現状分析

| 項目 | 現状 | 判断 |
|-----|------|------|
| Googleビジネス写真 | 88枚 | 量は十分・種類の最適化が必要 |
| 口コミ件数 | 30件 | 40〜50件がティッピングポイント |
| 口コミ依頼できる患者 | 14〜15人 | 全員から取れれば44〜45件→今週達成可能 |
| LP症例追加 | 可能 | Google投稿→LP→SNSの3連動が作れる |

---

## MEO競争の構造

```
大手・資金力あり：PPC入札↑ → 表示回数↑ → 口コミ↑ → 信頼↑ → 集客↑
江本院の勝ち筋  ：「量」ではなく「深さ」＝超ロングテール症例で戦う
```

### 軸②：「リアル」で差別化（AI洪水への対抗）

| 要素 | 具体的な方法 |
|-----|-----------|
| 実際の姿勢比較写真 | ビフォーアフターを毎週1枚投稿 |
| 患者の手書きコメント | 来院後に一言書いてもらい写真に撮る |
| 具体的な数字 | 「体重-4kg」「しびれが8→2に減少」 |
| 日付・期間 | 「3ヶ月・12回の施術で」 |
| 患者の職業・年代 | 「40代・事務職・2人の子育て中」 |
| 患者の言葉をそのまま引用 | 「こんなに変わるとは思いませんでした」 |

### 軸③：口コミを30件→50件へ

#### 個別LINE依頼テンプレート

```
○○さん、こんにちは。江本です。

先日は施術にお越しいただきありがとうございました。
（〇〇が楽になったとおっしゃっていたこと、
 私もとても嬉しかったです）

一つお願いがあるのですが、
もし体の変化を感じていただけているようでしたら、
Googleの口コミに一言いただけますか？

「どんな症状で来たか」
「来てみてどうだったか」

この2点だけで大丈夫です。

↓こちらから書けます
[Googleマップ口コミリンク]

お忙しいところ恐れ入りますが、
よろしくお願いします。
```

#### 送るタイミング

| タイミング | 効果 |
|----------|------|
| 施術直後に「体が変わった！」と言った瞬間 | その場でスマホで書いてもらえる確率が最高 |
| 施術翌日のLINE | 体の変化がまだ続いている段階 |
| 3回目の来院後 | 「通い続けてよかった」という実感が出る頃 |

#### カスタマイズ例

| 患者の記憶 | カスタム文例 |
|---------|-----------|
| 腰痛が1回で楽になった | 「腰が軽くなったとおっしゃっていたこと」 |
| 長年の症状が改善 | 「長年悩まれていた〇〇が改善されてきたこと」 |
| 喜んでいた表情が印象的 | 「あの施術後の笑顔が印象に残っています」 |

## LP症例コンテンツの構造

### 1ページ1症例で量産

```
URL例：
lp.emoto01.com/cases/desk-work-lower-back-numbness
lp.emoto01.com/cases/spinal-stenosis-recovery
lp.emoto01.com/cases/postpartum-pelvis-pain
```

### 各症例ページの構成

```
H1：【症例】50代女性・デスクワーク・右腰から足のしびれが3ヶ月で改善

① 症状の詳細（患者が検索するような自然な言葉で書く）
② 来院前の状態・他院での経緯
③ 原因の説明（図や写真があれば尚良し）
④ 施術の流れと変化
⑤ 患者の声（できれば写真付き）
⑥ 同じ症状の方へのメッセージ
⑦ CTA（LINE相談・予約）
```

## 写真88枚の最適化

| 写真の種類 | 理想的な枚数 | 目的 |
|---------|-----------|------|
| ビフォーアフター姿勢比較 | 20枚以上 | 実績証明・信頼 |
| 施術中の様子 | 10枚以上 | 「怖くない」安心感 |
| 院内・外観 | 10枚以上 | 来院前の不安解消 |
| 先生の顔写真・表情 | 10枚以上 | 人柄・信頼感 |
| 患者の笑顔（許可取得済） | 5枚以上 | リアルな実績 |
| 手書きの感謝コメント写真 | 5枚以上 | AIにない「本物感」 |

## Claude Codeでの効率化

症例コンテンツの量産フロー：

```
患者情報（年齢・症状・結果）を箇条書きで入力
↓
Claudeがテンプレートに沿った投稿文を生成
↓
確認・微修正して投稿
↓
Google投稿・LP・Instagram・LINEの4媒体分を同時生成
```
