## 「大切な人を紹介したくなる院」の深掘り

### 紹介が起きる瞬間

**タイミング：施術後に自然発生する**

同じ悩みを持つ人を思い浮かべてもらい、その人を連れてくる。
親子での来院が最も多い。

**患者さんの実際の言葉（そのままコンテンツに使える）：**
```
「私が楽になったから、いっぺん見てもらったら？」
「ここは間違いないから」
「痛みがない優しい施術で、すごく楽になって、
 先生もしっかり話聞いてくれるから一回行ってみたら？」
```

**「本当に良くなるの？」という疑いへの先生の返し：**
```
「ずっとそればかり勉強しているプロですから」
```
→ 自信と根拠が両方入った言葉。キャプション・固定投稿に使える。

---

### 院の空気感を表すキーワード

```
「お茶しにきているような感覚」
「肩肘張らずに来てくれている」
「めっちゃマイナスなことは言わない」
「基本は前向きなことしか言わない」
```

整体院なのに「お茶しに来る」感覚。
これをデザイン・映像・言葉で再現することが
「話しやすい先生」「紹介したくなる院」の核になる。

---

### 忘れられないエピソード（コンテンツの最重要素材）

**末期癌のお父さんの話**

| 要素 | 内容 |
|------|------|
| 紹介のきっかけ | 娘が元々通院していた |
| 来院時の状態 | 末期癌・息も絶え絶え |
| 変化 | 腰の痛みがなくなり・息もしやすくなった |
| 行動の変化 | 自分で病院に通えると思い車を買った |
| 大切な補足 | **癌が治ったわけではない** |

**なぜこのエピソードが最強か：**
- 娘→父という「大切な人への紹介」がそのまま体現されている
- 「癌が治ったわけではない」という正直さが信頼を生む
- 「車を買った」という具体的な行動変化が想像しやすい
- 重症者に「自分にも希望がある」と思わせる

**使用方法：患者さんは出演しない。先生が語るリール形式で使用。**

台本イメージ（60〜90秒）：
```
「今でも忘れられない患者さんがいます。

娘さんがずっと通ってくれていて、
あるときお父さんを連れてきてくれました。

末期癌で、息も絶え絶えの状態で。

正直、私にできることは限られています。
癌を治すことはできない。
ただ、腰の痛みをとることはできる。
息をしやすくすることはできる。

施術を続けていくうちに、
お父さんの腰の痛みがなくなっていきました。
息も、少しずつしやすくなって。

ある日こう言ってくださったんです。
『自分で病院に通えそうやから、車買おうと思う』

その言葉が、忘れられなくて。

癌は治っていない。
でもその人の生活が、少し広がった。

私がやりたいのは、そういうことなんやと思います。
どんな状態でも、諦めなくていい場面がある。」
```

活用媒体：Instagram リール・YouTube・LINE配信コラム

## 来てほしくない客層（自然に遠ざけるメッセージ設計）

直接「来ないで」とは言わない。
「こういう人が来る院です」を発信することで、
合わない人が自分で判断して来なくなる。

| 遠ざけたい特徴 | 発信での対処法 |
|-------------|-------------|
| 身体が治らないと思っている | 「治そうという気持ちがあれば変われる」を繰り返す |
| でも・だけど・どうせの否定言葉 | 「前向きに取り組んでくれる方と一緒に治していきます」 |
| こちらを信頼しようとしない | 「信頼して任せてくれる方と一緒に治していきます」 |
| 治療院巡り自慢をする | 触れない。来てほしい人に向けた発信だけをする |
| すぐに治ると思っている | 「3ヶ月で改善」の期間感覚を丁寧に伝えるコンテンツ |
| 10代〜20代前半の女性 | ターゲットの年代（30代後半〜）の症例を中心に出す |

## 「紹介率93%」を証拠化するコンテンツ

### ① 対談動画の切り抜き活用
対談動画の中で「院を知ったきっかけ」を聞いたとき、
「紹介してもらった」と答えている部分を切り抜いてリールにする。

```
活用方法：
- Q「当院を知ったきっかけは？」→ A「友人に紹介してもらいました」
- この部分だけを30秒以内に切り抜き
- テロップ：「患者様の〇割が紹介でご来院」を重ねる
- 複数の対談から集めて1本にまとめるバージョンも有効
```

### ② 親子・夫婦2ショット投稿
一緒に来院している親子・夫婦に撮影協力をお願いしてInstagramに掲載。

```
撮影のお願いのタイミング：
- 施術後の会計時に「よければ写真撮らせてもらえますか？」

投稿フォーマット：
- 2ショット写真
- キャプション例：「〇〇さんのご紹介でお母様も来てくださいました」
- ハッシュタグ：#寝屋川整体 #家族の健康 など
```

**この投稿が持つ効果：**
- 「家族に紹介したくなる院」が視覚で証明される
- 「自分の大切な人も連れて行きたい」という感情を引き出す
- 紹介のハードルが下がる
