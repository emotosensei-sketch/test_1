## ブランドカラー

**アンバー（琥珀色）をベースに統一する**

| 用途 | カラーコード |
|------|------------|
| メインカラー | #E8922A |
| サブカラー（深み） | #B5651D |
| ベース | 白（#FFFFFF）|
| テキスト | 濃いこげ茶（#3D2009）|

**選定理由：**
- ほがらか・温かみ → オレンジの明るさを持つ
- 話しやすい人 → 黄みがかった温かみで威圧感ゼロ
- 専門性・他院との違い → 深みがあり軽くならない／整体院でほぼ使われていない色

## 「紹介率93%」を証拠化するコンテンツ

コンセプト「大切な人を紹介したくなる院」を裏付ける投稿を積み上げる。

### ① 対談動画の切り抜き活用
対談動画の中で「院を知ったきっかけ」を聞いたとき、
「紹介してもらった」と答えている部分を切り抜いてリールにする。

```
活用方法：
- Q「当院を知ったきっかけは？」→ A「友人に紹介してもらいました」
- この部分だけを30秒以内に切り抜き
- テロップ：「患者様の〇割が紹介でご来院」を重ねる
- 複数の対談から集めて1本にまとめるバージョンも有効
```

### ② 親子・夫婦2ショット投稿
一緒に来院している親子・夫婦に撮影協力をお願いしてInstagramに掲載。

```
撮影のお願いのタイミング：
- 施術後の会計時に「よければ写真撮らせてもらえますか？」
- 「Instagramで紹介させていただきたいので」と一言添える

投稿フォーマット：
- 2ショット写真
- キャプション：「○○さんのご紹介で、お母様も来てくださいました」
- または「ご夫婦でのご来院、ありがとうございます」
- ハッシュタグ：#寝屋川整体 #家族の健康 など
```

**この投稿が持つ効果：**
- 「家族に紹介したくなる院」が視覚で証明される
- 「自分の大切な人も連れて行きたい」という感情を引き出す
- 紹介のハードルが下がる（みんなやっている）
