# MEO戦略

## 基本方針
- 大手の広告費に対抗するため「量」ではなく「深さ」で戦う
- 超ロングテール症例コンテンツで飽和した市場を避ける
- AIコンテンツが氾濫する中で「リアル」で差別化する

## 現状と目標
| 指標 | 現状 | 目標 |
|-----|------|------|
| Google口コミ | 30件 | 50件 |
| Googleビジネス写真 | 88枚 | 種類を最適化 |
| 週間投稿数 | 0本 | 2本 |

## 口コミ依頼LINE（個別送信テンプレート）
```
○○さん、こんにちは。江本です。

先日は施術にお越しいただきありがとうございました。
（〇〇が楽になったとおっしゃっていたこと、私もとても嬉しかったです）

一つお願いがあるのですが、Googleの口コミに一言いただけますか？

「どんな症状で来たか」「来てみてどうだったか」この2点だけで大丈夫です。

↓こちらから書けます
[Googleマップ口コミリンク]
```

**ベストタイミング：施術直後・翌日・3回目来院後**

## Google投稿テンプレート

