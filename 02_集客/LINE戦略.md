# LINE戦略

## アカウント構成
| LINE | 役割 |
|------|------|
| ハブLINE | 入口・メインLINEへの誘導のみ |
| 新規メインLINE | 教育・予約獲得・来院前フォロー |
| 既存患者LINE（新） | 予約管理・再来院促進・価格帯別配信 |
| 既存患者LINE（旧） | チラシ予約専用に転用 |

## 新規導線
```
Instagram・広告 → ハブLINE → 新規メインLINE
→ 分岐①：今すぐ予約 → 新規_予約直行_4通
→ 分岐②：腰チェック → 新規_腰チェック_7通
→ 予約確定 → 来院前確認 → 前日リマインド → 来院 → 既存患者LINEへ移行
```

## ラベル設計

### 新規用
- `新規_予約済` / `新規_来院済` / `新規_配信停止`
- `新規_前日リマインド送信済`

### 既存患者用
- `既存患者` / `6000円患者` / `5000円患者`
- `4月_予約済` / `4月_来院済` / `4月_配信停止`
- `メンテ患者` / `メンテ_フォロー開始` / `メンテ_停止`

### チラシ用
- `チラシ流入` / `チラシ_LINE予約` / `チラシ_WEB予約`
- `チラシ_主訴_腰痛` / `チラシ_主訴_頭痛` / `チラシ_主訴_首こり` / `チラシ_主訴_膝痛`

## 運用ルール
1. 新規LINEと既存患者LINEは必ず分ける
2. 予約が入ったら教育シナリオを止める
3. 来院後は必ず既存患者LINEへ移す
4. 既存患者LINEでは価格帯を混ぜない
5. 前日リマインドは手動一斉配信で対応
6. メンテ患者向けは来院2日後・21日後の2通のみ自動配信

## メンテ患者自動配信
- 1通目：来院2日後
- 2通目：来院21日後
- 長期離脱（60日・90日以上）：月1回手動一斉配信
